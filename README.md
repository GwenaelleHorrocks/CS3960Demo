1 PROBLEM STATEMENT
Data cleaning is essential for data analysis and exploration. The
missing values, inconsistent data and human errors have been a huge
challenge that many data analysts have had to overcome. Even with
the number of years spent researching solving these issues, it is still
a complicated task to prep and clean data. This is due to the large
majority of data cleaning being reliant on human judgment which
can introduce a whole new set of hurdles. [1] Inaccurate data can
cause businesses and companies to make bad decisions and draw
false conclusions that harm them. Cleaning data must extend to
more than just static data since real-time data and stream data also
need to be accurate and complete for research and constant data. [5]
Data profiling is a huge advantage in aiding the cleaning of data.
It is the process of assessing the data quality to detect what issues
arise in the data. It is a timely process but it is vital for the goal of
preforming analysis on clean and accurate data. [2, 3] With the use
of openclean, these issues are mitigated by offering a centralized
system that cleans the data and profiles. [4] This is an open source
python library that has an amalgamation of other libraries that are
well celebrated for data cleaning. Implementing and preforming a
demonstration of openclean will be my central goal in this project.
